🧠 Java Data Types

Java splits data types into 2 main categories:
1. Primitive Data Types

These are the basic building blocks. No drama, no objects, just raw values. 
          
          Integer Types:-
          | Type    | Size    | Example         |
          | ------- | ------- | --------------- |
          | `byte`  | 1 byte  | -128 to 127     |
          | `short` | 2 bytes | small integers  |
          | `int`   | 4 bytes | default integer |
          | `long`  | 8 bytes | large numbers   |
          Example:
          int age = 25;
          long population = 8000000000L;
          System.out.println("Byte Min: " + Byte.MIN_VALUE);
          System.out.println("Byte Max: " + Byte.MAX_VALUE);

          System.out.println("Short Min: " + Short.MIN_VALUE);
          System.out.println("Short Max: " + Short.MAX_VALUE);

          System.out.println("Int Min: " + Integer.MIN_VALUE);
          System.out.println("Int Max: " + Integer.MAX_VALUE);

          System.out.println("Long Min: " + Long.MIN_VALUE);
          System.out.println("Long Max: " + Long.MAX_VALUE);

          Floating Point Types:-
          | Type     | Size    | Precision            |
          | -------- | ------- | -------------------- |
          | `float`  | 4 bytes | ~6-7 digits          |
          | `double` | 8 bytes | ~15 digits (default) |
          Example:
          float price = 99.5f;
          double pi = 3.1415926535;

          System.out.println("Float Min: " + Float.MIN_VALUE);
          System.out.println("Float Max: " + Float.MAX_VALUE);

          System.out.println("Double Min: " + Double.MIN_VALUE);
          System.out.println("Double Max: " + Double.MAX_VALUE);

          Character Type:-
          Example:
          char grade = 'A';

          System.out.println("Char Min: " + (int) Character.MIN_VALUE);
          System.out.println("Char Max: " + (int) Character.MAX_VALUE);


          Boolean Type:-
          Example:
          boolean isActive = true;







2. Non-Primitive (Reference) Data Types

These are where Java starts acting like a “real programming language” and not just a calculator.

They store references (addresses), not actual values.

Common Types:

->String
->Arrays
->Classes
->Interfaces

Example:
String name = "Ganesh";
int[] numbers = {1, 2, 3};




Key Differences (Primitive vs Non-Primitive)

| Feature | Primitive     | Non-Primitive    |
| ------- | ------------- | ---------------- |
| Stores  | Value         | Reference        |
| Memory  | Stack         | Heap             |
| Size    | Fixed         | Dynamic          |
| Example | `int`, `char` | `String`, Arrays |



🧩 Important Notes (Interview Bait)

int is the default integer type

double is the default decimal type

char uses Unicode (2 bytes), not ASCII

String is not primitive (yes, Java likes to trick beginners)


🧠 One-Line Summary

Primitive = actual value
Non-primitive = address pointing to value
